# 🎓 Student Performance Tracker ERP

A full-stack Flask web application for managing students, subjects, marks, and performance analytics with authentication, dashboards, and reports.

---

## 👤 Author

**Vineela Kommu**  
Student ERP Flask Project (2026)

---

## 🚀 Features

- 👨‍🎓 Student Management (Add / Edit / Delete)
- 📚 Subject Management
- 📝 Marks Entry System
- 📊 Analytics Dashboard (Topper, Average, Performance)
- 📄 Student Report Cards
- 🔐 Login / Register System
- 🌙 Dark Mode Support
- ⚙ Settings (Profile Update + Change Password)

---

## 🛠 Tech Stack

- Python (Flask)
- Flask-SQLAlchemy
- SQLite Database
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Jinja2 Templates
